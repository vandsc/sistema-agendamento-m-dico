# Sistema de Agendamento Médico

Este projeto consiste no desenvolvimento de uma aplicação web para agendamento de consultas médicas, utilizando arquitetura com frontend e backend separados.

## Acesso ao sistema

O sistema está disponível online através do link:

🔗 https://sistema-agendamento-m-dico.vercel.app

## Tecnologias utilizadas

- Frontend: Vue.js (Vite)
- Backend: Node.js + Express
- Banco de Dados: MongoDB Atlas
- Deploy Frontend: Vercel
- Deploy Backend: Render

## Funcionalidades

- Cadastro de usuários
- Login com autenticação
- Agendamento de consultas
- Painel do usuário

## Estrutura do projeto

- `/frontend` → Interface do usuário
- `/backend` → API e regras de negócio

## Observações

O backend pode apresentar um pequeno tempo de resposta inicial devido ao plano gratuito do Render.
